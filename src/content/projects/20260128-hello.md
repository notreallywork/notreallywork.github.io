## Stack
Next.js 14 + Tailwind + Markdown. Static export to GitHub Pages. Custom domain `notreally.work`.

## Concept
Corporate directory aesthetics hosting chaotic experiments. File permissions as UI. Amber phosphor on CRT black. Everything is monospace. Nothing is rounded.
