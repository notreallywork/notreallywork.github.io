Shader Glitch 04

Attempting noise displacement at 3am. Works on desktop only.

#webgl#three.js#vibe-coding
Source Code↗Live Demo↗

Objective

Generate Perlin noise displacement in WebGL using Three.js. The goal was to create an interactive visualization where the user could manipulate noise parameters in real-time.

Result

Works in Firefox. Safari throws OES_texture_float_linear error. Chrome works but performance varies wildly depending on GPU.

What went right:

  • Noise generation is actually pretty
  • UI controls are responsive
  • Learned a lot about texture formats

What went wrong:

  • Safari doesn't support OES_texture_float_linear
  • Mobile performance is... bad
  • It's 4am and I'm hungry

Blockers

  • Safari WebGL limitations
  • No fallback for unsupported extensions
  • Coffee supply depleted

Next Steps

Probably abandon this and start something new. The cycle continues.
