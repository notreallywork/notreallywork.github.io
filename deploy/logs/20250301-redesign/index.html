Redesigning again

#design#brutalism#decisions

Changed the link color from blue to red. Now everything looks like an error message. I love it.

The decision

Was using #0066CC for links. Standard, accessible, boring. Switched to #FF0000. Alarming, aggressive, memorable.

Accessibility concerns

Red links can be problematic for colorblind users. But:

  1. All links are underlined (WCAG compliant)
  2. The red is distinct from the black text
  3. Hover states provide additional feedback

Also, this whole site is intentionally abrasive. Accessibility is important, but so is having a personality.

Other changes

  • Removed all border-radius (was using 2px, now 0px)
  • Increased border thickness from 1px to 1px (kept it the same, but thought about changing it)
  • Switched from system font stack to... the same system font stack

The truth

Most of these changes are procrastination. I should be writing content, but tweaking CSS feels productive.
